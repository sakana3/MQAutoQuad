# MQAutoQuad
AutoQuad for Metasequoia 

メタセコ用のAutoQuadプラグインです。  
面貼りが少し早くなるかも？

- 使い方  
ツールメニューからAutoQuadを選び四角形を貼りたいなって所でクリックします。
込み入った形状では上手く四角形が選ばれない時もありますのでその時は諦めましょう。
